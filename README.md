## 📌 Project Overview

This project is a **premium interactive dashboard** built using Streamlit to analyze Netflix content data.
It provides deep insights into content distribution, trends, genres, and key contributors like actors and directors.

The dashboard is designed with a **professional UI/UX approach**, inspired by modern BI tools like Power BI and Tableau.

---

## 🎯 Key Features

* 📊 Interactive Filters (Year, Country, Genre, Type)
* 🌍 Global Content Distribution Map
* 🎬 Movies vs TV Shows Analysis
* 🎭 Genre Insights (Top Categories)
* ⭐ Ratings Distribution
* 🎬 Top Directors & Actors Analysis
* 📈 Content Trend Over Time
* 📊 Growth of Netflix Content

---

## 🛠️ Tech Stack

* **Frontend & Backend:** Streamlit
* **Data Processing:** Pandas
* **Visualization:** Plotly
* **Deployment:** Streamlit Cloud
